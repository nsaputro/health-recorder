Google Integration
Setup Guide

To enable Google sync:

  1. Open Google Cloud Console
  2. Enable Google Health API, Google Sheets API, and Google Drive API
  3. Create OAuth 2.0 credentials → Web application
  4. Add authorized redirect URI: http://<ha-ip>:8099/auth/google/callback
  5. Set credentials in addon options (⚙️ Configuration tab in HA)
  6. Restart addon, then click Connect Google Account above
